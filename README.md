## Installation 🔐

- Download repository
- Load schema.sql and seeds.sql through command line or Sequel Pro
- Create .env file
- SET DB_NAME to 'employees_db'
- SET DB_USER to 'YOUR-MYSQL-USERNAME'
- SET DB_PASSWORD to 'YOUR-MYSQL-PASSWORD'
- npm install
- node index.js
